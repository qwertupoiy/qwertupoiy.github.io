프로젝트 개요

경제 뉴스를 자동으로 수집해 요약하고, 감성/핫토픽을 분석해 한 화면에 보여주는 서비스입니다.
분석 파트(FastAPI)와 서비스 파트(Spring Boot)를 분리해 ML 파이프라인을 바꿔도 서비스는 안정적으로 유지되도록 설계했습니다.
MongoDB로 사용자 관심사를 저장해 개인화 추천도 제공합니다.

주요 기능

뉴스 수집

경제 카테고리 기사 주기적 수집

요약/감성

FastAPI에 요청해 요약/감성/키워드 뽑기

대시보드

실시간 키워드, 감성 추이, 유튜브 댓글 분석 결과 노출

개인화 추천

MongoDB에 저장된 관심사 기반 도서/영상 추천

주요 화면

기술 스택

  • Service : Spring Boot
  • Analysis : FastAPI (Python)
  • DB : MongoDB Atlas
  • Data : Pandas, Transformers
  • Auth : 세션/쿠키 기반 로그인

특징

수집 → 분석 → 시각화 Spring ↔ FastAPI 연동 개인화 추천
